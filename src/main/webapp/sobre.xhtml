Sobre o Gerenciador de Tarefas

O Gerenciador de Tarefas é uma aplicação construída com Javax, PrimeFaces e Hibernate para ajudar equipes a acompanhar atividades do dia a dia. Ele permite criar, editar, filtrar e concluir tarefas de forma simples, mantendo o foco apenas no que ainda precisa ser feito.

Alguns destaques:

  • Cadastro rápido com validações claras para evitar dados inconsistentes.
  • Filtro dinâmico por responsável, prioridade, situação e palavras-chave.
  • Interface responsiva pensada para desktop e dispositivos móveis.
  • Integração com banco de dados PostgreSQL, garantindo persistência das tarefas.

Este projeto serve tanto como um exemplo de arquitetura JSF moderna quanto como um ponto de partida para soluções personalizadas de gestão de atividades. Conheça mais trabalhos em github.com/pedroleonez.
